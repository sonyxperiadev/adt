adt.widgets.grid

Module implementing a smart grid to store widgets. Part of the Analytics Dashboard Tools.

Requires: d3@v4 adt.widgets

flexible

Makes the grid flexible, that is, it rescales all contained widgets when the window size is changed. Note that this method overwrites windows.onresize.

add

adt.widgets.grid.Grid.add(widget, column, row, width, height)

Adds a widget to the specified row and column. If the specified cells are outside the grid or the cell is already occupied, it does not add the widget and returns null. The widget does not need to be rendered while adding, and the grid also sets positioning and size automatically. Note that the rows are measured from the top.
argtypedescription
widgetadt.widgets.WidgetWidget to add.
columnnumberColumn index of the cell to add widget to.
rownumberRow index of the cell to add widget to.
widthnumberNumber of columns to occupy.
heightnumberNumber of rows to occupy.
returndescription
adt.widgets.WidgetThe added widget if cells are available and free, null pointer othetwise.

get

returndescription
